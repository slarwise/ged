# ged - A good editor

This text was written with ged!!! Oopa gangman style.

## Quick start

```sh
go install github.com/slarwise/ged@latest
ged <filename>
```

Vimish keymaps:

- Normal mode:
  - i: Enter insert mode
  - space: Enter command mode
  - q: quit
- Insert mode:
  - escape: Enter normal mode
  - Other keys: Write
- Command mode:
  - w: write file
  - x: write file and quit
  - escape: Enter normal mode

CTRL-C quits `ged` in all modes.
